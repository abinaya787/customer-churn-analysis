# Customer Churn Analysis using Python

## Project Overview
This project analyzes customer churn data of over 7,000 customers to identify patterns and factors influencing churn behavior.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Steps
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Identifying churn patterns
- Data visualization

## Key Insights
- Month-to-month customers show higher churn
- Higher monthly charges increase churn risk
- Low tenure customers are more likely to leave
