# Customer Churn Analysis using Python

## Project Overview
This project analyzes customer churn data of 7,000+ customers to understand key factors influencing customer attrition and to generate actionable business insights for improving retention.

---

## Objectives
- Clean and preprocess customer data
- Perform Exploratory Data Analysis (EDA)
- Identify key factors affecting churn behavior
- Visualize patterns and trends in customer data
- Provide data-driven business recommendations

---

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn

---

## Key Insights
- Customers on month-to-month contracts have higher churn rates
- Higher monthly charges increase likelihood of churn
- Customers with low tenure are more likely to leave

---

## Business Impact
The analysis helps businesses identify high-risk customers and implement targeted retention strategies to reduce churn rate and improve customer lifetime value.
